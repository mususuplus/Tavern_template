# money 正文美化正则

这些条目用于酒馆“正则表达式替换”。它们只做显示美化，不修改 MVU。

建议启用范围：仅显示时替换 / 只影响 AI 输出。不要用于发送给 AI 的提示词。

这版不依赖额外 CSS 导入，直接用内联样式和浏览器原生 `title` 提示，先保证一定能看见效果。

## 金额高亮

名称：[money]金额高亮

查找正则表达式：`/(?<!["'=])([¥￥]\s*\d[\d,]*(?:\.\d+)?|\d+(?:\.\d+)?\s*(?:万|亿)?\s*(?:资金|元|块))(?![^<]*>)/g`

替换为：
```html
<span style="display:inline;border:1px solid rgba(240,200,108,.45);background:rgba(240,200,108,.14);color:#f8d879;border-radius:4px;padding:0 .22em;font-weight:700;text-shadow:0 0 10px rgba(240,200,108,.26);">$1</span>
```

## 风险高亮

名称：[money]风险高亮

查找正则表达式：`/(?<!["'=])(极高风险|高风险|中风险|低风险|风险[:：]?\s*(?:极高|高|中|低)|警觉|曝光|反噬|失控|失败)(?![^<]*>)/g`

替换为：
```html
<span style="display:inline;border:1px solid rgba(224,90,82,.45);background:rgba(224,90,82,.14);color:#ffaca6;border-radius:4px;padding:0 .22em;font-weight:700;">$1</span>
```

## 成交进展高亮

名称：[money]成交进展高亮

查找正则表达式：`/(?<!["'=])(成交|已成交|已收款|完成交易|让步|动摇|加码|抬价|开价|屈服|资产化)(?![^<]*>)/g`

替换为：
```html
<span style="display:inline;border:1px solid rgba(85,212,137,.42);background:rgba(85,212,137,.13);color:#a9f5c8;border-radius:4px;padding:0 .22em;font-weight:700;">$1</span>
```
